# Project : Dungeon Minigame
This is a model in which the 'Player' has to reach the end of a room or 'Dungeon' while avoiding enemies and losing health. 
Features : 
1. Interactive User interaction.
2. Simple and clean UI.
3. Fast execution.

How to use :
1. Input either w, a, s or d for moving your character which starts in room 0.
2. If you lose all your health then you will have to run the code again and restart.
3. Make sure to remember the rooms which are safe and which are not safe.

Requirements :
Python 3.x
Jupyter Notebook

AI use :
I used AI to brainstorm a bug and a few ideas but main implementation were handled by me.
